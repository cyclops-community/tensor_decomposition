## Download and unpack the tensor

In the `saved-tensors` directory, run 
```bash
bash download.sh embedding 
```
to download.

## Add tuned model to google drive

Upload the file to [this link](https://drive.google.com/drive/u/0/folders/1Xxgzp15zalmz8eBf8oEW2BypzXSQ_6UF),
and add the fileid and filename in the download.sh.
